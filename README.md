# Simple eBanking System

A desktop banking application featuring a custom graphical user interface. This program manages financial transactions and user accounts by connecting a C# frontend to a MySQL database.

## Technologies Used
* C# (.NET Framework/Core)
* MySQL Database
* Windows Forms or WPF (Custom GUI)

## Prerequisites
* Visual Studio 2019 or newer
* MySQL Server
* MySQL Connector/NET

## Setup
1. Import the provided SQL schema into your MySQL instance to set up the database.
2. Update the connection string in the application code with your database credentials.
3. Build and run the project using Visual Studio.

## Key Functionalities
* User authentication (Login/Register).
* Account balance overview.
* Fund transfers between accounts.
* Transaction history logs.
* Profile management.

## Project Structure
* /Models - Data structures and database entities.
* /Views - Custom GUI forms and interface design.
* /Controllers - Logic for handling user input and database communication.
* /Resources - Icons and visual assets.
